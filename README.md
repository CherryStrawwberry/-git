# Все что я пока знаю:


## Команды:


Cd - перемещаться между директориями
cp - копировать из одной папки в другую (сначала указываем файл/файлы, потом папку)
Cat - открыть содержимое файла в консоли
git init - создать из папки репозиторий
Git add - добавить изменения для коммита
Git commit - сделать коммит
Git push - сделать закинуть коммиты в репозиторий
Touch - создать файл
Mkdir - создать папку
Rm - удалить файл
Rmdir - удалить папку (можно удалить только пустую)


## Флаги:


-r - удалить все из папки по очереди а затем саму папку (используем тогда rm, а не rmdir)
-m - сообщение к коммиту
-u - связывает локальную ветку с одноименной удаленной
-v - показать больше информации (например git remote -v)
